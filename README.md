# Luban Excel生成Skill

## 简介:
这个 Claude Code 技能帮助你快速配置和使用 Luban 数据表工具，生成 C#/Java/Go/Lua/TypeScript 等多种语言的代码和数据文件。


## 快速开始
1. 安装技能

> 将本仓库克隆到你的 .claude/skills/ 目录：


    ```
    cd ~/.claude/skills
    git clone https://github.com/qq790git/luban-skill
    ```

2. 使用技能
> 在 Claude Code 中输入 Luban 相关的问题即可触发技能：

```示例
"帮我创建一个物品表"
```


参考仓库：https://github.com/waduheck/claude-skill-luban
